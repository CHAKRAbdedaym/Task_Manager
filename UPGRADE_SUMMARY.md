# Upgrade Summary - Task Manager v2.0.0 (SaaS)

This document summarizes the changes made to transform the Task Manager from a single-shared-list app into a multi-user SaaS product.

## Key Features

1.  **JWT-Based Authentication**: Secure registration and login using industry-standard JWT.
2.  **Multi-Tenancy**: Complete data isolation. Users only see and manage their own tasks.
3.  **Task Richness**:
    *   **Priorities**: Low, Medium, High badges.
    *   **Due Dates**: Integrated calendar selection.
    *   **Categories**: Organize tasks by labels.
    *   **Search & Filtering**: Real-time search and dynamic filtering by status/priority.
    *   **Pagination**: Smooth handling of large task lists.
4.  **Premium UI/UX**:
    *   Modern **Glassmorphism** design system.
    *   **Responsive App Shell** with sidebar navigation.
    *   Animated transitions and micro-interactions.
    *   Empty states and loading skeletons.

## Technical Stack

*   **Backend**: Java 21, Spring Boot 3.5.x, JPA/Hibernate, PostgreSQL, JJWT.
*   **Frontend**: Angular 19, CSS3 (Modern features), Material Symbols.
*   **DevOps**: Docker, Docker Compose, Kubernetes.

## Infrastructure Changes

*   Added `APP_JWT_SECRET` environment variable for stateless authentication.
*   Updated Kubernetes secrets and deployment manifests.
